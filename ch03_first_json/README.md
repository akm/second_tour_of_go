# 初めてのJSON

## 予習してきてください

- JSONとは何か
- JSONで使える型の種類 

## :exclamation: セットアップ

1. ターミナルあるいはコマンドプロンプトで `ch03_first_json` に移動
2. `go mod init github.com/akm/second_tour_of_go/ch03_first_json` を実行
    - `github.com/akm` の部分は他の文字列に変更してもOKです。
        - Github等で管理するのであれば、自分の環境に合わせて変更してください
